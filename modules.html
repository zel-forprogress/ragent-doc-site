项目模块介绍#

bootstrap#

这是主业务模块。你大部分时间都在看它。

包 作用
user 登录、用户、权限
admin 仪表盘统计
knowledge 知识库、文档、chunk
ingestion Pipeline 入库
rag 问答核心
core 文档解析和分块

framework#

通用基础设施：

包 作用
convention 统一对象
exception 异常体系
web Web 工具、全局异常、SSE
mq RocketMQ 封装
idempotent 幂等
trace RAG Trace
context 用户上下文

infra-ai#

模型基础设施：

包 作用
chat Chat 模型
embedding Embedding 模型
rerank Rerank 模型
model 模型选择和故障转移
config 模型配置

mcp-server#

工具服务模块，模拟外部业务系统。

frontend#

React 前端，既有用户问答页，也有管理后台。

项目模块介绍#

bootstrap#

bootstrap 是主服务模块，里面包含大部分业务。

主要包：

包 说明
user 登录、用户管理、Sa-Token 权限
admin 管理后台统计
knowledge 知识库、文档、分块、入库任务
ingestion 文档入库 Pipeline
rag 问答、意图、检索、Prompt、MCP Client、Trace
core 文档解析和文本分块

framework#

通用能力模块：

包 说明
convention 统一 Result、ChatRequest、RetrievedChunk
exception 业务异常体系
web 全局异常处理和 SSE 工具
mq RocketMQ 生产者封装
idempotent 接口和消费幂等
trace RAG Trace 上下文
context 用户上下文

infra-ai#

AI 基础设施模块：

包 说明
chat 聊天模型客户端
embedding Embedding 模型客户端
rerank Rerank 模型客户端
model 模型选择、健康状态、故障转移
config 模型配置绑定
token Token 估算

mcp-server#

MCP 工具服务：

类 说明
MCPEndpoint /mcp HTTP 入口
MCPDispatcher JSON-RPC 方法分发
DefaultMCPToolRegistry 工具注册表
WeatherMCPExecutor 天气查询示例
TicketMCPExecutor 票务查询示例
SalesMCPExecutor 销售查询示例

frontend#

前端技术栈：

  • React 18。
  • Vite。
  • TypeScript。
  • Zustand。
  • React Router。
  • Radix UI。
  • Tailwind CSS。

前端 API 统一封装在：

frontend/src/services

状态管理在：

frontend/src/stores
